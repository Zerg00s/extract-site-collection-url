# SharePoint Site Collection URL Extractor

Extract site collection URLs from SharePoint URLs in bulk. Handles `/sites/`, `/teams/`, `/personal/`, and root sites.

[![Open App](https://img.shields.io/badge/Open%20App-0078d4?style=for-the-badge&logo=microsoftsharepoint&logoColor=white)](https://zerg00s.github.io/extract-site-collection-url)

![Screenshot](/img/Screenshot.png)

## Features

- Paste thousands of URLs, get site collections instantly
- Shows unique site collections with counts
- Highlights invalid URLs
- Copy results with one click

## Example

**Input:**
```
https://contoso.sharepoint.com/sites/Marketing/Shared Documents/Report.docx
https://contoso.sharepoint.com/teams/Sales/Documents/Q4.xlsx
```

**Output:**
```
https://contoso.sharepoint.com/sites/Marketing
https://contoso.sharepoint.com/teams/Sales
```
